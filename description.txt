Описание проекта Pygame 
Название игры/проекта: “Logic Functions” 

Проект представляет из себя игру-головоломку,
которая основана на двоичной логике (“не”, “и”, “или”). 
 
В проекте будет присутствовать несколько уровней, 
в которых для прохождения требуется из некоторых входных данных, 
представляющих True/False или 0/1, получить определённый результат. 
 
Так как этот проект основан на двоичной логике,
то в нём для построения данного потребуется размещать прямоугольные блоки “and”, “or”, “not”, “input”, “output”, 
соединяя их input/output сигналы между собой в зависимости от требуемого результата. 

В игре будет доступен режим песочницы, 
где можно создавать свои блоки, используя базовые или ранее созданные. 
Для работы такого кастомного блока его входные и выходные сигналы должны быть логически связаны. 
Вдобавок блоки можно будет редактировать, загружая предыдущее расположение блоков. 
 
В меню выбора уровней каждому уровню присвоена картинка и описание.
